# es6 babel boilerplate

This repository helps to setup the base structure for future projects.

## First things first

clone this repo, then run:

```sh
$ npm install
```

## How to
While develope use:

This active `nodemon`
```sh
$ npm run dev
```

To production:

```sh
$ npm run build
```

----------------

After install and test the new commands, you are ready to code.
